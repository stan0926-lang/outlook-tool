# Outlook Mail Tool - 配布用パッケージ

## 📦 内容物

- `OutlookMailTool.exe` - メインアプリケーション
- `README.txt` - 使用方法

## 🚀 使い方

### 1. 起動方法

`OutlookMailTool.exe` をダブルクリックして実行してください。

- コンソールウィンドウが開きます
- 自動的にブラウザが起動します
- `http://127.0.0.1:5000` にアクセスします

### 2. 初回起動時

初回起動時は空のデータベースが作成されます。
デモデータを使用する場合は、「データ取込」画面からデモデータをロードしてください。

### 3. 終了方法

コンソールウィンドウで `Ctrl+C` を押すか、ウィンドウを閉じてください。

## ⚠️ 注意事項

### システム要件
- Windows 10/11 (64bit)
- メモリ: 最低2GB推奨
- ディスク空き容量: 100MB以上

### セキュリティ警告
初回実行時にWindows Defenderやファイアウォールの警告が表示される場合があります。
「詳細情報」→「実行」を選択してください。

### ネットワーク
本アプリケーションはローカル環境でのみ動作します。
外部ネットワークへの接続は行いません。

### データの保存場所
データベースファイル `data.db` は、EXEファイルと同じフォルダに作成されます。
このファイルをバックアップすることで、データを保存できます。

## 🔧 トラブルシューティング

### ブラウザが開かない
- 手動で `http://127.0.0.1:5000` にアクセスしてください
- ポート5000が他のアプリケーションで使用されている可能性があります

### 「Python312.dll が見つかりません」エラー
- EXEファイルが破損している可能性があります
- 再度ダウンロード/コピーしてください

### データが消えた
- `data.db` ファイルが削除または移動されていないか確認してください
- バックアップから復元してください

## 📞 サポート

問題が発生した場合は、コンソールウィンドウに表示されるエラーメッセージを
確認してください。
